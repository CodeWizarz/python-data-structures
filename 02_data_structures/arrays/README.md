# Arrays

An array is a collection of items stored at contiguous memory locations.

## Implementation

Here is a simple implementation of an array in Python:

```python
# arrays.py

# Creating an array
arr = [1, 2, 3, 4, 5]

# Accessing elements
print(arr[0])  # Output: 1

# Modifying elements
arr[1] = 20
print(arr)  # Output: [1, 20, 3, 4, 5]
